# 💻 Code Project

O Code Project é um site educacional desenvolvido com o objetivo de introduzir conceitos básicos de programação de forma simples e acessível para iniciantes.

O projeto foi criado como parte de um itinerário escolar, com foco em apresentar o que é programação, sua importância no cotidiano e como dar os primeiros passos na área.

---

## 🚀 Tecnologias utilizadas

- HTML  
- CSS  

---

## 💡 Funcionalidades

- Explicação introdutória sobre programação  
- Apresentação de aplicações da programação no dia a dia  
- Conteúdo didático voltado para iniciantes  
- Navegação entre diferentes seções do site  

---

## 🎯 Objetivo do projeto

Facilitar o primeiro contato com a programação, tornando o aprendizado mais acessível e compreensível para pessoas que estão começando do zero.

---

## 🧠 O que aprendi

- Estruturação de páginas web com HTML  
- Estilização e organização visual com CSS  
- Organização de conteúdo para melhor experiência do usuário  
- Criação de projetos com foco educacional
- História da programação
